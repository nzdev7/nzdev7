## Quick Tips:

1. **For GitHub** - Use blue colors (#2563eb) - looks professional
2. **For Portfolio** - Use your brand colors
3. **Keep it readable** - Don't use too many different colors
4. **Test on dark mode** - Make sure colors work in both themes
5. **Stay professional** - Avoid bright neon colors
